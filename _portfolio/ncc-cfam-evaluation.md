This collaborative project with the **National Composites Centre (NCC)** evaluates nonplanar Continuous Fibre AM (CFAM) design tools for composite structures.

### Key Goals

- Assess CFAM design tools for complex geometries and load cases
- Benchmark print quality, fibre placement, and structural performance
- Provide feedback for improving CFAM design workflows and software tools

- **Completed**
